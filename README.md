**Project Overview**

This repository contains manual test cases written for testing a web-based Simple Calculator application.
The test cases are designed to validate the correctness of basic arithmetic operations and to ensure proper handling of valid and invalid user inputs.

**Application Under Test**

Application Name: Simple Calculator

URL: https://dunizb.github.io/sCalc/

Type: Web-based application

**Objective**

**The objective of this task is:**

*To verify basic arithmetic operations

*To validate decimal and negative number handling

*To ensure BODMAS (operator precedence) is followed

*To check error handling for invalid inputs such as division by zero

**Testing Type**

1.Manual Testing

2.Black-Box Testing

3.Functional Testing

**Features Tested**

1.Addition

2.Subtraction

3.Multiplication

4.Division

5.Decimal calculations

6.BODMAS rule

**Invalid input handling**

1.Division by zero
